xG Boosted

For fun and my own interest, I maintain an international soccer prediction model and odds. Please note these estimates are for demonstrative purposes and use at your own risk. The methodology will be posted on my blog in the near future and the github repo can be found here.

The page name is a mix between xG (expected goals - a metric in soccer analytics) and the XGBoost models.

Data Dictionary:

  • date - Date of Game
  • game - Teams playing
  • T1 - Probability that first team listed in column “game” wins
  • Draw - Probability that the game ends in a draw
  • T2 - Probability that second team listed in column “game” wins
